# Enfoques para Preguntas Iniciales

Este documento presenta **tres posibles enfoques** para el onboarding inicial de la plataforma de aprendizaje adaptativo. Cada enfoque representa un estilo distinto de interacción, permitiendo evaluar cuál encaja mejor con las expectativas del usuario final.

---

## 1. Enfoque Conversacional

Diseñado para sentirse como una charla ligera y natural, sin estructura rígida ni formato de cuestionario.

**Características:**

* Tono casual y cercano.
* Preguntas abiertas o semiabiertas.
* Enfoque en comodidad y baja fricción.
* Interpretación implícita de categorías cognitivas.

**Ejemplos de preguntas:**

* "Cuando algo te atrapa, ¿sos más de meter mano o investigar primero?"
* "¿Qué tipo de cosas te dejan pensando un rato después?"
* "Si te doy 10 minutos libres, ¿qué hacés sin pensarlo?"
* "¿Te entusiasma más entender cómo funciona algo o imaginar lo que podría existir?"
* "Si esta app fuera un parque de diversiones, ¿a qué atracción irías primero?"

**Ventajas:**

* Muy amigable.
* Sensación de personalización inmediata.

**Desventajas:**

* Interpretaciones más difusas.

---

## 2. Enfoque Storytelling

El onboarding se plantea como una pequeña narrativa inmersiva, donde el usuario toma decisiones dentro de un pequeño mundo ficticio.

**Características:**

* Atractivo e imaginativo.
* Menos preguntas, más decisiones.
* Ideal para captar atención al inicio.

**Ejemplos de preguntas:**

* "Imaginá que despertás en una sala futurista con tres puertas. ¿Cuál abrís?"
* "Hay un objeto frente a vos: un cuaderno, un botón rojo, una tableta luminosa o una lupa. ¿Cuál elegís?"
* "Tu objeto reacciona y te muestra algo intrigante. ¿Qué hacés? Analizás, probás, cuestionás o reinterpretás."
* "¿Qué habilidad inicial querés desbloquear: entender, crear, optimizar o explorar?"

**Ventajas:**

* Enganchante y divertido.
* Posiciona la experiencia como una aventura.

**Desventajas:**

* No todos los usuarios disfrutan de las narrativas.

---

## 3. Enfoque Gamificado

El onboarding es una serie de micro-interacciones visuales inspiradas en juegos casuales o tests interactivos.

**Características:**

* Selección mediante imágenes.
* Elecciones rápidas.
* Interacción visual inmediata.

**Ejemplos de interacciones:**

* Elegir una "tarjeta" que represente al usuario (ej. herramienta, pincel, cerebro, mapa).
* Seleccionar una imagen que genere curiosidad.
* Elegir un "superpoder" temporal para aprender.
* Deslizar hacia una opción más divertida.

**Ventajas:**

* Muy dinámico.
* Ideal para experiencias móviles.

**Desventajas:**

* Requiere más diseño visual.

---

## Comparación General

| Criterio                       | Conversacional | Storytelling | Gamificación |
| ------------------------------ | -------------- | ------------ | ------------ |
| Fricción inicial               | Muy baja       | Media        | Media-baja   |
| Nivel de inmersión             | Medio          | Alto         | Alto         |
| Requerimientos visuales        | Bajos          | Medios       | Altos        |
| Flexibilidad de interpretación | Alta           | Media        | Media        |
| Sensación de "entretenimiento" | Media          | Alta         | Alta         |
| Adecuado para usuarios serios  | Sí             | Depende      | Depende      |

---

## Recomendación para estudio de usuario

Para validar cuál enfoque usar, se sugiere:

* Probar los tres con usuarios reales.
* Medir tiempo de respuesta, engagement y claridad percibida.
* Analizar qué tipo de usuario prefiere cada enfoque.

Este documento sirve de base para definir qué estilo de onboarding potenciar en la versión final.
